Class zebra.ui.grid.GridCaption
extends: zebra.ui.grid.BaseCaption

Grid caption class that implements rendered caption. Rendered means all caption titles, border are painted as a number of views.

Parameters:
  • a <Array>

    caption titles. Title can be a string or a zebra.ui.View class instance
